# Security Policy

## Supported Versions

Only the latest version is actively supported with security updates.

| Version | Supported |
| ------- | --------- |
| latest  | ✅ |

## Reporting a Vulnerability

To report a security vulnerability, please open a [GitHub Security Advisory](https://github.com/keremsahin1/pickthebestcard/security/advisories/new) rather than a public issue.

We'll respond within 48 hours and aim to release a fix within 7 days for confirmed issues.
